# yejinchen-ai.github.io

Yejin Chen 的个人学术主页 / Personal academic homepage.

- **Website / 主页**: `https://yejinchen-ai.github.io`
- **Tech stack**: Static HTML + CSS, hosted on GitHub Pages.
- **Template inspiration / 模板参考**: 页面结构设计参考了 [Jian Wang's homepage](https://jianwang-cmu.github.io/)。

## 本地预览 / Local Preview

```bash
# 克隆仓库 / clone the repo
git clone git@github.com:yejinchen-ai/yejinchen-ai.github.io.git
cd yejinchen-ai.github.io

# 使用任意静态服务器在本地预览，例如 Python
python -m http.server 8000
```

在浏览器中打开 `http://localhost:8000` 即可预览当前 `index.html`。
